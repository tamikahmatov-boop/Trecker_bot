# 🤖 Crypto Bot

Telegram бот для мониторинга криптовалют с сигналами по изменению цены.

## 🚀 Деплой на Railway

1. Форкните репозиторий
2. На Railway создайте новый проект из GitHub
3. Добавьте переменные окружения (см. `.env`)
4. Деплой автоматический

## 📱 Команды

- `/start` - запуск
- `/status` - статус
- `📊 Статистика` - статистика
- `📈 X%` - установить порог
- `⏱ X мин` - установить период

## 🛠️ Локальный запуск

```bash
pip install -r requirements.txt
python main.py
